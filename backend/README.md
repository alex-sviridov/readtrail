# ReadTrail Backend API

Express.js backend API for the ReadTrail book tracking application.

## Features

- RESTful API for books and settings management
- Three authentication modes: No Auth, HTTP Header Auth, and Guest Mode
- In-memory database for testing
- Comprehensive input validation
- CORS support for frontend integration
- Full test coverage with Jest and Supertest

## Installation

npm install

## Configuration

Create a .env file based on .env.example

## Running the Server

Development: npm run dev
Production: npm start

## Testing

npm test
